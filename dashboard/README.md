# DashLite – SaaS Dashboard Template

DashLite is a modern **SaaS dashboard template built with Next.js, Tailwind CSS, and Base UI**.  
It provides a clean, scalable starting point for building **admin panels, SaaS platforms, CRM systems, and analytics dashboards**.

This template includes a responsive layout, sidebar navigation, dropdown menus, and reusable UI components.

---

## Features

• Modern SaaS dashboard layout  
• Responsive sidebar navigation  
• Light / Dark theme support  
• User profile dropdown  
• Notifications UI  
• Analytics & dashboard pages  
• Modular component architecture  
• Built with Tailwind CSS  
• Built for Next.js App Router

---

## Tech Stack

- **Next.js 16**
- **React**

- **Base UI**
